addSlidingTimeWindow

Description

Add a sliding window column to the data

Usage

addSlidingTimeWindow(data, dateCol, window, unit, dateFormat = "%m/%d/%Y",
  includeUntil = FALSE, relativeTime = TRUE, sample = 2147483647,
  keyCol = NA)

Arguments

data:

data.table to be modified.

dateCol:

The column name in data that will be used.

window:

The window length (numeric)

unit:

The window length unit (i.e. "second", "minute", "hour", "day", "week")

dateFormat:

provide date format for parsing. defaults to "%m/%d/%Y " see strtptime for more examples i.e. "%m/%d/%Y %I:%M:%S %p"

keyCol:

An optional key for the sliding window (NA as default)

includeUntil:

optional argument

relativeTime:

optional relative time boolean flag

sample:

optional maximal possible sample value (default to maxint)

Value

The new data
