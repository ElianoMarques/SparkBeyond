writeToFile

Description

Convert all data columns into a serializable text/primitive representation and write them to a tab separated file.

Usage

writeToFile(data, outputFile)

Arguments

data:

dataframe / data data.table to write.

Examples

grouped = groupBy(getData("titanic_train"), by =list("pclass","sex"))
writeToFile(grouped, "titanic_grouped.tsv")
