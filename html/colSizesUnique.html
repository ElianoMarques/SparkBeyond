colSizesUnique

Description

Count the unique number of items in each column

Usage

colSizesUnique(data)

Arguments

data

dataframe / data.table to be examined

Value

a summary of unique items counts per row

Examples

grouped = groupBy(getData("titanic_train"), by ="pclass")
