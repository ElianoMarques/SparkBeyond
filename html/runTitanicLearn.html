Description

Perform learn on SparkBeyond titanic example.

Usage

runTitanicLearn(configuration = "1", server_port = 9000)

Arguments

configuration

Configuration ID as string. 1 - use only J48, 2 - use RRandomForest, 3 - use defaultList.

server_port

the port to be accessed in the SparkBeyond API server. 9000 by default.

Value

SBmodel object containing the result of the learning

Examples

model = runTitanicLearn()
