sampleDataByClass

Description

Under/over sample data based on labels of columnName.

Usage

sampleDataByClass(data, columnName, labels = NA, desiredDistribution = NA)

Arguments

data

dataframe to be sampled

columnName

name of column with with labels by which to modify label distributions

labels

Optional. labels in columnName for which desiredDistribution will relate to. Default NA will assume equal distributions between all labels.

desiredDistribution

Optional. The requested distribution ratio of labels. Default NA will assume equal distributions between all labels.

Value

a sampled dataset with the requested distributions

Examples

hist(sampleDataByClass(getData("titanic_train"), "survived", c(0,1), c(3,1))$survived, plot = FALSE)$counts
